<H1>This</H1>

인스턴스 자신의 메모리를 가르킴

- 객체를 생성하게 되면 heap 메모리에 할당이 된다
  - 해당 instance의 주소값을 가지는 것이 참조 변수 -> 그 값이 참조 값\
- instance자체 내부에서 자기 자신의 memory를 가르키는 것이 this
- constructor에서 다른 constructor를 호출할 때 사용
- 자신의 주소 참조값 자체를 반환하는 경우 사용된다
